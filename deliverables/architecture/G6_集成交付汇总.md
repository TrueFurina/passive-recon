## 7. 架构核心冻结结论速查

- **四层架构**：L1 合规网关(GW/CE/AP/SC, 100% 自研) → L2 规划Agent(PL/DS/BL/XC) → L3 采集集群(C1 Web/C2 公众号/C3 小程序/C4 工商股权) → L4 知识库(N4/V4/EX)
- **北极星**：WNSR ≥ 100%
- **纯被动红线**：违规 = 0、封禁 = 0
- **算力调度**：A:B:C = 60:30:10、25min 回收、每 5min 看榜
- **三级人机审批**：操作方 → 主理人 → 评委
- **MVP 范围**：R1-R11 + R12 基础（F1-F16 中 P0 均属 MVP）
- **部署基线**：16C64G 单宿主机 Docker Compose（20 容器）；MySQL 8.0 主从 + Redis 7.2 哨兵 + RabbitMQ 3.13 + Neo4j 5.18
- **SLA**：可用性 ≥ 99.5%、P99 ≤ 2s、RTO ≤ 30min、RPO ≤ 15min
- **安全基线**：STRIDE 18 条威胁、IAM 2 种认证 + RBAC1 三级、数据 L1-L4 分级、密钥 K-01~K-05 五类、审计 5 维度 + 4 层不可篡改
